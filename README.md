# Business Intelligence Report Generator

An AI-powered business intelligence report generator for Recipe Labs that analyzes company websites and generates comprehensive business intelligence reports.

## Features

- **Automated Website Analysis**: Scrapes and analyzes company websites
- **AI-Powered Intelligence**: Uses Claude AI to generate comprehensive business intelligence reports
- **Professional Reports**: Creates beautifully formatted reports in Recipe Labs' signature style
- **Multiple Analysis Depths**: Choose from Standard, Comprehensive, or Deep Dive analysis
- **Secure Processing**: All processing happens securely in your browser

## Usage

1. Open `bi_report_generator.html` in a web browser
2. Enter the company website URL
3. Optionally provide the company name (or let AI extract it)
4. Enter your Anthropic API key
5. Select analysis depth
6. Add any additional context if needed
7. Click "Generate Business Intelligence Report"

## Requirements

- Modern web browser (Chrome, Firefox, Safari, Edge)
- Anthropic API key (get one from [console.anthropic.com](https://console.anthropic.com/))

## Report Sections

The generated report includes:

1. Executive Summary
2. Company Overview
3. Market Analysis
4. Competitive Intelligence
5. Operational Assessment
6. Financial Analysis
7. Digital Presence & Marketing
8. Strategic Assessment
9. Opportunity Analysis for Recipe Labs
10. Recommended Actions
